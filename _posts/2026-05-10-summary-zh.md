## [报告揭露中国 Claude API 灰产内幕](https://www.tomshardware.com/tech-industry/artificial-intelligence/chinese-grey-market-sells-claude-api-access-at-90-percent-off-through-proxy-networks-that-harvest-user-data) ⭐️ 8.0/10

一份新报告揭露，中国灰产中的 API 代理服务（即“中转站”）以低至官方一折的价格转售 Anthropic 的 Claude 模型访问权，但常常掉包为廉价模型并窃取用户数据。 这种做法给使用这些代理的开发者与企业带来了严重的安全与道德风险，包括数据窃取、模型掉包以及潜在的知识产权泄露，损害了 AI API 生态系统的信任。 这些代理通过盗刷信用卡、批量注册账号或拆分订阅套餐来降低成本，甚至雇佣低收入人群进行身份验证。它们还会采集用户的提示词与输出用于模型蒸馏。

telegram · zaihuapd · May 10, 01:48

**背景**: API 代理服务（即“中转站”）在中国很常见，用于绕过网络限制并降低访问 Claude 等国外 AI 模型的成本。然而，许多此类服务非法运营，未经授权转售访问权限，并从事模型掉包和数据窃取等欺诈活动。
